

## How to setup virtual cam

![[OBS-Sources-SS.png|400]]
1. find sources window
2. click "+"
3. select "Media Source"
4. unselect "Local File"
5. Add video feed endpoint into "input" textbox (for example: "rtsp://unctalos.student.rit.edu:8554/camera")
![image](/technical/architecture/OBS-SS.png)
6. find "Controls" window
7. click "Start Virtual Camera"
